# CT 肺结节诊断专家共识 (2023) — 知识摘要

## 一、肺结节定义与分类

### 1.1 定义
肺结节是指肺内直径 ≤ 3cm 的类圆形或不规则形病灶，影像学表现为密度增高影，可单发或多发，边界清晰或不清晰。

### 1.2 按大小分类
- **微小结节**：直径 < 5mm
- **小结节**：直径 5mm ~ 10mm
- **结节**：直径 10mm ~ 30mm
- **肿块**：直径 > 30mm（超出结节范围，需警惕恶性）

### 1.3 按密度分类
- **纯磨玻璃结节 (pGGN/pGGO)**：CT 上呈均匀云雾状密度增高影，不掩盖血管和支气管纹理
- **混合磨玻璃结节 (mGGN/mGGO/部分实性)**：含磨玻璃成分和实性成分两种密度
- **实性结节 (Solid Nodule)**：均匀软组织密度，完全掩盖血管和支气管

### 1.4 按数量分类
- **单发/孤立性肺结节 (SPN)**：肺内单个结节
- **多发性肺结节**：肺内 ≥ 2 个结节

## 二、恶性风险评估

### 2.1 临床危险因素
| 因素 | 低风险 | 高风险 |
|------|--------|--------|
| 年龄 | < 40 岁 | > 50 岁 |
| 吸烟史 | 从不吸烟 | 年吸烟量 > 400 支年 |
| 家族史 | 无肺癌家族史 | 一级亲属肺癌史 |
| 职业暴露 | 无 | 石棉、氡、铀等接触史 |
| 既往肿瘤史 | 无 | 有胸部以外恶性肿瘤史 |

### 2.2 CT 影像恶性征象

**典型恶性征象：**
1. **分叶征 (Lobulation)**：结节边缘呈分叶状或扇贝状轮廓，≥ 3 个弧形凸起，深度 ≥ 2mm
   - 对恶性结节的阳性预测值约 75-85%
2. **毛刺征 (Spiculation)**：结节边缘呈放射状条索影
   - 最特异的恶性征象，特异度 > 90%
3. **胸膜凹陷征 (Pleural Indentation)**：结节与胸膜之间的线状或三角形致密影
4. **血管聚集征 (Vascular Convergence)**：周围血管向结节聚拢
5. **空泡征 (Vacuole Sign)**：结节内小泡状低密度区（直径 1-3mm）
6. **支气管截断/充气征**

**典型良性征象：**
1. 钙化（层状、中心致密、爆米花样）
2. 脂肪密度
3. 边缘光滑、无分叶
4. 卫星灶（结核球特征）

### 2.3 风险评估模型

**Lung-RADS 分类标准：**

| 类别 | 描述 | 结节特征 | 恶性风险 | 建议 |
|------|------|----------|---------|------|
| 1 | 阴性 | < 1% | 常规随访 |
| 2 | 良性 | 实性结节 < 6mm；新发 < 4mm | < 1% | 12 个月随访 |
| 3 | 可能良性 | 实性结节 6-8mm；亚实性 ≥ 6mm(GGO) | 1-2% | 6 个月随访 |
| 4A | 可疑恶性 | 实性结节 ≥ 8mm；新发 6-8mm | 5-15% | 3 个月随访/PET-CT |
| 4B | 高度可疑 | 实性结节 ≥ 15mm；新发 ≥ 8mm | > 15% | PET-CT/活检 |
| 4X | 高度可疑 | 有额外特征(毛刺/生长快) | > 15% | 增强CT/PET-CT/活检 |

## 三、随访策略

### 3.1 实性结节随访建议
| 结节大小 | 低风险人群 | 高风险人群 |
|----------|-----------|-----------|
| < 6mm | 无需常规随访 | 可 12 个月后随访 |
| 6-8mm | 6-12 个月随访，稳定后 18-24 个月复查 | 6-12 个月随访，稳定后 18-24 个月复查 |
| > 8mm | 3 个月随访/PET-CT/活检 | 3 个月随访/PET-CT/活检 |

### 3.2 磨玻璃结节随访建议
| 结节类型/大小 | 建议 |
|--------------|------|
| pGGN < 6mm | 无需随访 |
| pGGN ≥ 6mm | 6-12 个月随访，稳定后每 2 年随访至 5 年 |
| mGGN < 6mm | 6-12 个月随访，稳定后每 2 年随访至 5 年 |
| mGGN ≥ 6mm 实性成分 < 6mm | 3-6 个月随访，后每 2 年随访至 5 年 |
| mGGN 实性成分 ≥ 6mm | 3-6 个月随访/PET-CT/活检 |

## 四、CT 技术规范

### 4.1 扫描参数
- **层厚**：≤ 1mm 薄层扫描（建议 0.625-1.25mm）
- **重建间隔**：与层厚一致
- **窗宽/窗位**：肺窗（1500/ -500 HU）、纵隔窗（350/ 40 HU）
- **管电压**：100-120 kVp
- **自动管电流调制**：推荐使用

### 4.2 结节测量
- **二维测量**：最大层面长径 × 短径
- **三维测量**：体积测量（更准确，尤其随访对比时）
- **倍增时间**：恶性实性结节通常 30-400 天；良性结节通常 > 400 天
- **磨玻璃结节**：倍增时间可能较长（> 800 天），需长期随访

## 五、处理路径

### 5.1 孤立性肺结节处理流程
```
发现肺结节(≤3cm)
    ↓
评估临床风险(年龄/吸烟/家族史)
    ↓
CT 特征评估(大小/密度/边缘/形态)
    ↓
风险分层(Lung-RADS)
    ├─ 低风险(<5%) → 影像随访
    ├─ 中风险(5-30%) → 短期随访/PET-CT
    └─ 高风险(>30%) → 活检/手术切除
    ↓
随访中结节稳定≥2年 → 降低随访频率
随访中结节增大/密度增高 → 升级处理
```

### 5.2 手术指征
- 实性结节 ≥ 15mm 且高度可疑恶性
- 随访中结节明显增大（体积增加 ≥ 25%）
- PET-CT SUVmax > 2.5
- 活检证实恶性

## 六、报告书写规范

### 6.1 报告必备要素
1. **结节描述**：位置（肺叶/肺段）、大小（三维）、密度、边缘
2. **定量指标**：CT 值(HU)、体积、倍增时间（若有既往对比）
3. **风险评估**：Lung-RADS 类别、恶性概率
4. **处理建议**：随访时间、下一步检查、是否需要活检

### 6.2 报告模板示例
```
【检查所见】
右肺上叶后段(S2)见一 12mm × 10mm 混合磨玻璃结节，
实性成分约 6mm，边缘可见分叶征及毛刺征，
CT 值约 -350 HU，胸膜轻度牵拉。
余肺野未见明确结节和实变影。

【诊断意见】
右肺上叶混合磨玻璃结节（Lung-RADS 4A类），
建议 3 个月后薄层 CT 随访。

【定量：结节体积 1.2cm³，平均 CT 值 -350HU】
```
