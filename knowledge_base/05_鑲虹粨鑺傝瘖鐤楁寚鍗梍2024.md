# 肺结节诊疗指南 (2024) — 知识摘要

## 第一章：筛查与早期发现

### 1.1 高危人群筛查标准
**肺癌筛查高危人群定义（符合以下任一）：**
1. 年龄 50-80 岁
2. 吸烟史 ≥ 20 包年（或戒烟 < 15 年）
3. 有肺癌家族史（一级亲属）
4. 有职业暴露史（石棉、氡、铬、镍、砷等）
5. 有 COPD 或肺纤维化病史
6. 既往有恶性肿瘤史

### 1.2 筛查方法
- **推荐**：低剂量 CT (LDCT)，参数 ≤ 1.5mSv
- **不推荐**：胸部 X 线平片（灵敏度不足）
- **不推荐**：痰细胞学、血清肿瘤标志物作为独立筛查手段

### 1.3 筛查频次
- 高危人群：每年 1 次 LDCT
- 发现结节后的随访频率依据 Lung-RADS 分级决定

## 第二章：诊断方法

### 2.1 影像学诊断
- **增强 CT**：评估结节血供、纵隔淋巴结
- **PET-CT**：≥ 8mm 结节的代谢评估，SUVmax ≥ 2.5 提示恶性可能
- **MRI**：胸壁/膈肌侵犯评估
- **动态增强 MRI**：鉴别良性结节（结核球/错构瘤）

### 2.2 非手术活检
| 方法 | 适应症 | 诊断准确率 | 并发症 |
|------|--------|-----------|--------|
| 经支气管针吸活检(TBNA) | 中央型病变/纵隔淋巴结 | 75-90% | 气胸 1-5% |
| 经皮肺穿刺活检(PTNB) | 周围型病变 | 85-95% | 气胸 10-30% |
| 支气管镜活检 | 中央型病变 | 70-90% | 出血 1-3% |
| 电磁导航支气管镜 | 周围型微小结节 | 70-85% | 低于 PTNB |

### 2.3 肿瘤标志物
| 标志物 | 灵敏度 | 特异度 | 说明 |
|--------|--------|--------|------|
| CEA | 40-60% | 70-85% | 非特异性，腺癌升高明显 |
| CYFRA21-1 | 50-70% | 80-90% | 鳞癌敏感性较高 |
| SCC-Ag | 30-50% | 85-95% | 鳞癌特异度较高 |
| ProGRP | 60-80% | 90-95% | 小细胞肺癌特异指标 |
| NSE | 50-70% | 70-85% | 小细胞肺癌辅助诊断 |

## 第三章：病理分类

### 3.1 肺癌病理类型（WHO 2021 分类）

**非小细胞肺癌 (NSCLC) — 约占 85%：**
1. **腺癌 (Adenocarcinoma)**：最常见（~40%）
   - 贴壁型 (Lepidic)、腺泡型 (Acinar)、乳头型 (Papillary)
   - 微乳头型 (Micropapillary)、实性型 (Solid)
2. **鳞状细胞癌 (Squamous Cell Carcinoma)**：~25%
   - 角化型、非角化型、基底样型
3. **大细胞癌 (Large Cell Carcinoma)**：~5%
4. **腺鳞癌 (Adenosquamous)**：~2%

**小细胞肺癌 (SCLC) — 约占 15%：**
- 小细胞癌、混合性小细胞癌

**神经内分泌肿瘤：**
- 类癌 (典型/非典型)、大细胞神经内分泌癌

### 3.2 分子分型（NSCLC 必检基因）
| 驱动基因 | 发生率（腺癌） | 靶向药物 |
|----------|---------------|---------|
| EGFR | 40-60%（亚洲） | 吉非替尼/奥希替尼/厄洛替尼 |
| ALK 融合 | 3-7% | 克唑替尼/阿来替尼 |
| ROS1 融合 | 1-2% | 克唑替尼/恩曲替尼 |
| BRAF V600E | 1-3% | 达拉非尼+曲美替尼 |
| MET 扩增/14跳读 | 3-5% | 卡马替尼/特泊替尼 |
| RET 融合 | 1-2% | 塞尔帕替尼 |
| KRAS G12C | 10-15% | 索托拉西布/阿达格拉西布 |
| NTRK 融合 | < 1% | 拉罗替尼/恩曲替尼 |

## 第四章：治疗原则

### 4.1 分期治疗策略

**NSCLC 分期治疗：**
| 分期 | 治疗策略 |
|------|----------|
| 0 期 (原位癌) | 局部切除/消融 |
| IA 期 (T1N0M0) | 手术切除（肺叶/段切除） |
| IB-IIB 期 | 手术 + 辅助化疗（高危因素时） |
| IIIA 期 | 新辅助化疗+手术/同步放化疗 |
| IIIB 期 | 同步放化疗 ± 免疫治疗 |
| IV 期 | 系统治疗（靶向/免疫/化疗） |

**SCLC 分期治疗：**
| 分期 | 治疗策略 |
|------|----------|
| 局限期 (I-III) | 同步放化疗 ± 预防性脑照射 |
| 广泛期 (IV) | 化疗 + 免疫治疗 |

### 4.2 结节随访管理
- 稳定结节（≥ 2 年不变）：降低随访频率或停止随访
- 增大结节（体积增加 ≥ 25%）：升级检查或干预
- 新发结节：按肺结节处理流程重新评估

## 第五章：预后

### 5.1 分期与生存率（NSCLC）
| 分期 | 5 年生存率 |
|------|-----------|
| IA1 | 90-97% |
| IA2 | 85-90% |
| IA3 | 80-85% |
| IB | 70-80% |
| IIA | 55-65% |
| IIB | 45-55% |
| IIIA | 25-35% |
| IIIB | 10-20% |
| IV | < 5% |

### 5.2 预后影响因素
- **有利因素**：早期发现、腺癌、EGFR/ALK 驱动基因阳性
- **不利因素**：晚期、小细胞癌、KRAS 突变、PS ≥ 2
- 早期发现将 5 年生存率从 < 5% 提升至 > 85%
