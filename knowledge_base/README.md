| 文件 | 类型 | 用途 | 来源 |
|------|------|------|------|
| 01_肺叶分段解剖图谱.md | Markdown | 肺叶/肺段解剖结构、淋巴结分区 | 自行整理自 Gray's Anatomy、CT 解剖图谱 |

## 用途说明

这些文件将被 RAG 系统向量化后用于：
- Agent 诊断推理时的知识检索
- 报告生成时的术语和编码引用
- 追问时的解剖/病理知识支持

## 向量化后存储
每个文件将被切分为 500-1000 token 的 chunk，通过 Embedding 向量化后存入 pgvector。
